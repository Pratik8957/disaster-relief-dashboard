# 🚑 Disaster Relief Dashboard

A responsive web application designed to provide essential disaster management information in one place. The dashboard helps users access emergency contacts, view safe shelters, monitor weather conditions, explore disaster locations on an interactive map, and stay informed through alerts.

This project was developed using React and focuses on creating a clean, user-friendly interface with practical features that can be extended for real-world disaster response systems.

---

## Features

- Interactive disaster map using React Leaflet
- Live weather information with OpenWeather API
- Emergency contacts section
- Safe shelters information
- Blood donor directory
- Missing persons section
- Latest news page
- Search functionality
- Responsive layout for desktop and mobile
- Modern dashboard interface

## Getting Started

Clone the repository

```bash
git clone https://github.com/Pratik8957/disaster-relief-dashboard.git
```

Go to the project directory

```bash
cd disaster-relief-dashboard
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```
